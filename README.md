# 🥋 Coding Dojo - VSCode Agents

**Master the Power of GitHub Copilot Agents!**

Welcome to the Coding Dojo, your hands-on learning experience for building intelligent, agentic workflows with GitHub Copilot in Visual Studio Code. In this workshop, you'll build a real-world application while discovering how AI agents can revolutionize your development process.

## 🎯 What You'll Learn

This dojo teaches you how to leverage **GitHub Copilot Agents** to:
- ✨ Automate complex development tasks
- 🚀 Build features faster with AI assistance
- 🎮 Create interactive applications with modern frameworks
- 🧠 Understand agentic AI patterns and best practices

## 📱 The Project: Social Bingo

You'll build **Soc Ops**, a fun, interactive Social Bingo game designed for in-person mixers and events. Players match colleagues or participants to questions and race to get 5 in a row—all while learning about GitHub Copilot Agents!

**Features:**
- 🎲 Dynamic bingo board generation
- ❓ Customizable question sets
- 🎯 Real-time game state management
- 📱 Responsive mobile-friendly design

## 🚀 Quick Start

Before you begin, make sure you have:
- [Node.js 22](https://nodejs.org/) or higher
- GitHub Copilot enabled in VS Code

Then run:

```bash
npm install
npm run dev
```
